TripSplit is **100% free** with no hidden costs, subscriptions, or premium features.

## What's Free

✅ **All Core Features**
- Create unlimited groups
- Add unlimited members
- Add unlimited bills
- Custom split ratios
- Member exclusion from splits
- Expense calculations
- Settlement suggestions

✅ **All Advanced Features**
- Real-time data synchronization
- Expense statistics and analytics
- Category breakdowns
- Mark settlements as paid
- Export/print settlement summaries
- Member-specific expense views

✅ **All Storage**
- Free cloud storage via jsonstorage.net
- LocalStorage backup
- No storage limits

✅ **All Hosting**
- Free GitHub Pages hosting
- No server costs
- No bandwidth limits

1. **jsonstorage.net** - Free JSON storage (no registration needed)
